# Playbook — 슬라이드 덱 (PPT · 협업 제안서 · 마케팅 제안서 · IR)

> 슬라이드형 산출물 작업 시 blueprint 작성 전에 읽는다.
> `standards/design-standards.md` 공통 기준 위에 얹히는 전용 기준이다.

---

## 0. 이 산출물의 본질

덱은 **주장 문서**다. 슬라이드 1장 = 주장 1개 + 근거. 장식은 주장을 빛나게 할 때만 가치가 있다.
먼저 용도를 확정하라 — 기준이 달라진다:

- **발표용**(스크린에 띄우고 말로 보완): 텍스트 최소, 큰 타이포, 슬라이드당 15초 이해.
- **열람용**(제안서·이메일 첨부, 혼자 읽음): 밀도 높게, 본문 완결형 문장, 각주·부록 허용.
협업/마케팅 제안서는 대부분 **열람용**이다. 불명확하면 사용자에게 묻는다.

## 1. 캔버스 규격

- 16:9 — **1920×1080px 고정** (4:3 요청 시 1440×1080).
- 마진: 상하좌우 96px(열람용은 80px까지 허용). 마진 안쪽이 콘텐츠 영역.
- 12컬럼 그리드(거터 24px)를 기본으로 좌우 분할·3분할 레이아웃을 구성.

## 2. 타이포 스케일 (1920×1080 기준)

| 역할 | 발표용 | 열람용 |
|---|---|---|
| 표지 제목 | 88~120px | 80~100px |
| 섹션 디바이더 제목 | 64~88px | 56~72px |
| 슬라이드 헤드라인 | 44~56px | 40~48px |
| 핵심 숫자(KPI 등) | 100~180px | 80~140px |
| 본문 | 28~36px | 22~28px |
| 캡션·출처·각주 | 20~24px | **18px 하한** |

- 헤드라인은 **결론형 문장**으로: "시장 현황" ✕ → "이 시장은 3년 내 2배가 됩니다" ✓.
- 슬라이드당 본문 텍스트 상한: 발표용 40단어 / 열람용 120단어. 넘치면 슬라이드를 쪼갠다.

## 3. 덱 아키텍처

### 협업/마케팅 제안서 표준 뼈대
1. 표지(제안 한 줄 요약 + 로고 + 날짜)
2. 익스큐티브 서머리(제안 전체를 1장으로 — 열람용 필수)
3. 문제/기회 정의(시장 데이터·타겟 인사이트)
4. 제안 본론(무엇을·어떻게 — 2~5장, 장당 주장 1개)
5. 근거(실적·사례·데이터·포트폴리오)
6. 실행 계획(일정·역할 분담·프로세스)
7. 조건(가격·범위 — 표로)
8. 팀/회사 소개(신뢰 요소: 이력·수상·레퍼런스 로고)
9. 클로징 CTA(다음 스텝 + 연락처)

### 공통 규칙
- 3장 이상 이어지는 본론 앞에는 목차, 파트 전환마다 섹션 디바이더.
- 마스터 요소 통일: 페이지 번호, 로고 위치, 푸터 — 전 슬라이드 동일 좌표(오토레이아웃/컴포넌트화).
- 데이터는 표보다 차트로, 차트는 메시지 1개만(강조 시리즈 1개만 Accent 컬러, 나머지는 회색).
- 같은 역할 슬라이드는 같은 레이아웃을 재사용한다 — 덱의 일관성이 곧 신뢰감이다.
  (레이아웃 종류는 덱 전체에서 4~6종 이내로 제한)

## 4. 배경·비주얼 전략

- 배경 기본: 단색(밝은 회백/딥 네이비 등) + 브랜드 컬러 포인트. 슬라이드 배경에 생성 이미지를
  전면으로 까는 것은 표지·섹션 디바이더에서만 허용(16:9 ≈ 1536×1024 생성 이미지와 매칭 가능).
- 본문 슬라이드의 생성 이미지는 **콘텐츠 영역 안의 오브젝트**(제품컷, 컨셉 비주얼)로만.
- 아이콘·다이어그램·차트는 전부 Figma 벡터. 화살표/프로세스/조직도를 생성 이미지로 뽑지 않는다.
- 스크린샷·실적 이미지는 P 에셋으로 요구(모서리 라운드+섀도우로 통일 처리).

## 5. 셀프 QA 추가 항목

- [ ] 슬라이드마다 헤드라인이 결론형인가 (제목만 읽어도 스토리가 이어지는가)
- [ ] 표지→클로징 스토리라인이 논리적으로 끊기지 않는가
- [ ] 마스터 요소(페이지 번호·로고·푸터) 좌표가 전 슬라이드 동일한가
- [ ] 레이아웃 종류가 6종 이내이고 같은 역할엔 같은 레이아웃인가
- [ ] 열람용: 본문 ≥ 22px, 각주 ≥ 18px / 발표용: 본문 ≥ 28px
- [ ] 차트가 메시지 1개만 강조하는가 (전부 컬러면 실패)
